# Titanic Dataset - Exploratory Data Analysis

## Objective
Perform Exploratory Data Analysis to identify patterns, relationships, and key insights from the Titanic dataset.

## Dataset Information
- 891 rows
- 12 columns
- Features include Age, Sex, Fare, Pclass, Survived, etc.
